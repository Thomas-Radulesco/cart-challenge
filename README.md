# 🛒 Cart Challenge — React + TypeScript E‑Commerce Demo

A modern, responsive e‑commerce cart application built with **React**, **TypeScript**, **Vite**, and **Styled‑Components**, featuring real API integration, category filtering, product pages, cart management, and a clean UI.

This project was completed as part of an **Engineering Challenge**.

## 🛠 Setup & Run Instructions

### 1. Install dependencies
npm install
### 2. Start development server
npm run dev


### 3. Build for production
npm run build

### 4. Preview production build
npm run preview

## 🧪 Testing

- **Run tests (watch mode):** `npm run test`
- **Run tests with UI:** `npm run test:ui`
- **Run tests once with coverage report:** `npm run coverage`

Coverage outputs a **text summary** in the terminal and an **HTML report** in `coverage/index.html` (open in a browser for a detailed view).
